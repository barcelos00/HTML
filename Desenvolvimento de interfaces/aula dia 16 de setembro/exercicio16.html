Seletor de ID

O seleto rde ID é aquele que aplica a regra css a cada componente html que tenha um atributo no componente HTML. Esse seletor de id deve ser aplicado a um unico elemento HTML, haja vista, que o id em tese deve ser o unico. Além disso, esse seletor terá prioridade máxima sibre os demais seletores (de classe e de tipo)

Neste exemplo estamos utilizando seletores de id para estabilizar o documento html.
