  • Angular
      Angular é um framework front-end criado pelo Google. Ele serve para desenvolver aplicações web de página única (SPA – Single Page Applications).
    • Diferente de usar apenas HTML, CSS e JavaScript puro, o Angular oferece:
      • Componentes: divide a aplicação em partes reaproveitáveis.
      • Data Binding: liga os dados do código à interface de forma automática.
      • Diretivas: adicionam comportamento extra aos elementos do HTML.
      • Roteamento: permite trocar de “páginas” sem recarregar o site todo.
      • TypeScript: o Angular é escrito em TypeScript (um superconjunto do JavaScript que adiciona tipagem).
  • Vue.js
      Framework progressivo para interfaces web, criado por Evan You (ex-Google). Focado em ser simples de aprender e leve
    • Caracteristicas:
      • Muito fácil de integrar em projetos já existentes.
      • Usa componentes também, mas de forma mais flexível que Angular.
      • Tem reatividade forte: quando os dados mudam, a interface muda automaticamente.
      • Tem reatividade forte: quando os dados mudam, a interface muda automaticamente.
  • Blazor C#
      Framework da Microsoft para criar aplicações web modernas. Com sua linguame principal sendo C#. Possui o foco de permitir que desenvolvedores .NET façam aplicações web sem precisar usar só JavaScript.
    • Diferente de usar apenas HTML, CSS e JavaScript puro, o Angular oferece:
      • Você escreve C# no lugar de JavaScript.
      • Funciona de dois modos:
        1. Blazor Server → o código roda no servidor e envia atualizações para o navegador.
        2. Blazor WebAssembly → o código C# roda direto no navegador usando WebAssembly.
      • Integra-se muito bem com ASP.NET e todo o ecossistema Microsoft.

    Web semantica:

    A web semantica é uma estenção da eweb atual, onde a informação possui signifiacdo e permite que computadores e pessoas possam trabalhar em conjunto. As tags da web semantica ajudam os mecanismos de busca na tarefa de criar repositórios e recuperar informaçõe. As principais tags são: <header>, <nav>, <main>, <section>, <article>, <aside> e <footer>.

    Neste exemplo estamos usando a web semantica em todo o docuemtno html
