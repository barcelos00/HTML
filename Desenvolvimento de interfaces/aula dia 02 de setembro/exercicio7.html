Exemplo 7 HTML Uso das tags <ol>, <ul> e <li>

As tags <ol>, <ul> representão listas em html, e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada a tag <ul> representa uma lista não ordenada.

Em ambas as listas para representr os elementos da lista usamos a tag <li>

Além disso, estas listas podem conter outras lista de forma alinhadas.

Etapas para Desenvolver um Site (Lista Ordenada)

  1. Planejamento
    1. Definir objetivo do site(Lista Ordenada).
    2. Identificar o publico alvo.
    3. Criar um mapa do site(site map).
  2. Desing
    1. Criar wireframes e prototipos.
    2. Definir a paleta de cores e tipografia.
    3. Desenvolver um layout no Figma ou Adobe XD.
  3. Desenvolvimento
    1. Configurar o ambiente de desenvolvimento.
    2. Codificar o front-end (html, JavaScript e css).
    3. Implementar o back-end (Node.js, PHP, C#, Java).
  4. Teste
    1. Testar a responsividade em diferentes dispositivos.
    2. Realizar testes de usabiliade.
    3. Corrigir os bugs identificados.
  5. Publicação
    1. Escolher um provedor de hospedagem.
    2. Configurar o dominio SSL.
    3. Fazer o deploy do site.
